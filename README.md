# Social Media Video Downloader Bot 🤖

بوت تليجرام لتحميل فيديوهات من السوشل ميديا بدون علامة مائية.

## المنصات المدعومة

- TikTok
- Instagram
- Twitter / X
- YouTube
- Facebook
- Pinterest
- Snapchat
- Threads
- Likee

## التثبيت

### 1. إنشاء بوت تليجرام

- افتح [@BotFather](https://t.me/BotFather) في تليجرام
- أرسل `/newbot` واتبع التعليمات
- احفظ الـ **Bot Token**

### 2. تثبيت المشروع

```bash
npm install
```

### 3. إعداد ملف البيئة

عدّل ملف `.env` وضع التوكن:

```
BOT_TOKEN=123456789:ABCdefGhIjKlMnOpQrStUvWxYz
```

### 4. تشغيل البوت

```bash
# تشغيل عادي
npm start

# تشغيل مع إعادة تشغيل تلقائي عند التعديل
npm run dev
```

## طريقة الاستخدام

1. افتح البوت في تليجرام
2. أرسل `/start`
3. الصق رابط أي فيديو من المنصات المدعومة
4. انتظر قليلاً وسيصلك الفيديو بدون علامة مائية

## ملاحظات

- الحد الأقصى لحجم الفيديو: **50 ميجابايت** (حد تليجرام)
- الفيديوهات الخاصة لا يمكن تحميلها
- البوت يستخدم **Cobalt API** كمحرك أساسي للتحميل (مجاني بدون مفتاح)
- يمكن إضافة **RapidAPI** كبديل احتياطي (يحتاج مفتاح API)
